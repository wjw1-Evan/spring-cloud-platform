# Spring Cloud Platform

This project is initialized from an Ant Design Pro template and adapted for the Spring Cloud Platform microservices demo. Follow the quick guide below to use the frontend.

## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## Provided Scripts

This project provides scripts to help you quick start and build the web project, check code style and run tests.

Scripts provided in `package.json`. It's safe to modify or add additional script:

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
